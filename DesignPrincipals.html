Key Ideas

  • Accessability
  • Consistancy
  • Affordances
  • Feedback
  • Constraints
  • Visability

Design Approaches

  • Double Diamond: discover, define, develope & deliver
  • User centered design focues on user needs
  • Activity focuses on certain function
  • Genius is when the designer takes control
  • Human Centered design helps solve probelems by using real people
  • Getting user input early in the design

Interactivity Principals

  • Feedback should be given to the user quickly
  • Every button and click should be meaningful
  • 3 clicks to reach anyware in the site
  • Empathy for user errors

Cognitive Principals

  • Keep users attention with visuals
  • The perception is based on how information is presented and what order
  • Make sure to have memorable features and design
  • Make it clear what should be pressed to get to certain functions of the site

Compleity

  • Should be simple to use
  • There will always be some level of compleity no matter how much you simplify
  • The user shouldn't have to think
  • Users shouldn't need a tutorial
  • Learn by themselves though trial

Gender and Language Principals

  • Different colors apeal to different genders
  • Genders want different labeling and look for differnt features
  • Certain languages read different directions

Mobile/Other Interfaces Principals

  • Have different layouts for smaller or larger screens
  • Have a setup for touch screen users
  • Include the option to change certain features to fit their needs
